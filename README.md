# Bus Fuel Consumption Prediction

This project predicts fuel consumption using a Decision Tree model and provides a user-friendly GUI using Tkinter.

## Features
- Machine Learning model (Decision Tree)
- Data preprocessing and cleaning
- GUI for user input
- Fuel and cost estimation

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Tkinter

## How to Run
1. Install required libraries:
   pip install pandas scikit-learn

2. Run the file:
   python fuel_prediction.py
